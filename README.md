- [UK online voting API](https://github.com/reicraftscodes/voting-system-spring-boot)
- [UAS Lifecycle Management API](https://github.com/reicraftscodes/uas-lifecycle-management)

##  Core Skills Demonstrated
**Backend**
- Java, Spring Boot 2 & 3 (Spring JPA, Spring JDBC, Spring Security, Spring Actuator)
- Logging: SLF4J (Simple Logging Facade for Java), Log4J
- Relational Database Management System and Optimisation (Indexing) using MySQL
- API documentation
- Junit Test, Test driven development, BDD
- Dependency Management Tool: Gradle & Maven
- Sonarquabe / Sonarlint

**Frontend**
- React (Runs on Node.js)
- Bootstrap CSS, Material UI React Libary
- JavaScript Chart.js
- Thymeleaf
- E2E testing Cypress

**DevOps Practices**
- Git, GitHub, Gitlab
- Jenkins, Red Hat OpenShift
- Heroku, Docker, AWS
